    • mysql8安装

    下载地址

    mysql-8.0.11-winx64.zip

    下载下来时zip格式,直接解压到想放置的目录下

    配置环境变量

    • 把bin目录配置到环境变量下

    • 配置my.ini文件,mysql8默认没有my.ini,自己创建一个my.ini文件

    • 初始化数据库

      在MySQL安装目录的 bin 目录下执行命令：

      mysqld –initialize –console

    • 安装服务

      • mysql可以安装多个服务,用名字区分,安装命令:mysqld –install [服务名]
      • 启动MySQL服务:net start mysql
      • 停止MySQL服务:net stop mysql
      • 卸载MySQL服务:sc delete mysql

    • 更改密码

      执行命令:mysql -u root -p

      提示输入密码,输入刚才生成的密码登陆mysql.

      使用下面的命令修改密码:

      ALTER USER ‘root‘@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘新密码’;

    至此,mysql8安装完成.
