    koa2安装

    • 使用npm install koa-generator -g全局安装koa生成器

    • 创建项目:koa2 -e project ,-e表示使用ejs模板引擎

    • 安装依赖: cd project && npm install
    • 启动app:SET DEBUG=koa* npm start project
      • 启动项目相关命令
      • npm start(修改代码,不自动更新)
      • npm run dev
      • npm run prd
      • npm test

    koa路由

    • new Router
    • instance
      • .get|put|post|patch|delete|del =>Router
      • .routes= function
      • .use([path],middleware)=>Router
      • .prefix(prefix)=>Router
      • .allowedMethods[options]=>function
      • .redirect(source,destination,[code])=>Router
      • .url([name,params,[options]])=>String|Error
      • .param(param,middleware)=>Router
    • static
      • .url(path,params)
